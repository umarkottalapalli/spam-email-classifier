# spam-email-classifier

This project uses **Machine Learning and Natural Language Processing (NLP)** to classify emails as **Spam** or **Not Spam (Ham)**.

The model analyzes email text and predicts whether the message is unwanted spam or a legitimate email.

Spam emails are a major problem in communication systems.  
This project builds a **machine learning model** that automatically detects spam messages using **text classification techniques**.

The system processes email text, extracts useful features, and predicts the category of the email.

---

## Features

- Email text preprocessing
- Stopword removal and tokenization
- TF-IDF feature extraction
- Machine learning classification
- Model evaluation and accuracy measurement

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Natural Language Processing (NLP)

---

## Dataset

The dataset contains labeled email messages.

Labels:

- **Spam** → Unwanted promotional or malicious messages  
- **Ham** → Legitimate emails

Example:


| "Congratulations! You won a free iPhone. Click here now!" | Spam |
| "Meeting scheduled for tomorrow at 10 AM." | Ham |
